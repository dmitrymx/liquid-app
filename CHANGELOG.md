# Changelog

Все заметные изменения в проекте документируются здесь.

## [1.9.0] - 2026-06-02

### Новые функции
- Добавлен раздел **«Глубокие твики» (System Tweaks)** с иконкой `tune`
- Внедрен **Деблоатер Windows (UWP App Uninstaller)** для безопасного удаления встроенных UWP пакетов и кнопкой восстановления базовых приложений
- Внедрены **Твики телеметрии и конфиденциальности** (WER, Cortana, Ads ID, Ink) с полной возможностью отката
- Внедрен **Редактор контекстного меню Проводника (Shell Extensions)** для отключения захламляющих меню пунктов
- Внедрен **Редактор Hosts** с кнопкой быстрой блокировки серверов сбора данных Microsoft
- Добавлен инструмент **«Сброс сети» (Network Reset)** на страницу Сетевых утилит для сброса DNS, TCP/IP и Winsock
- В раздел автозагрузки добавлена поддержка **Планировщика задач (Task Scheduler)** для глубокого обнаружения скрытых автозапусков

## [1.8.0] - 2026-06-02

### Исправления багов
- **Критическое исправление приватности Firefox:** Изолирована очистка cookies, предотвращающая потерю паролей (`logins.json`) и закладок/истории (`places.sqlite`)
- **Безопасная очистка Chromium:** Реализовано динамическое определение профилей браузеров (Chrome, Edge, Yandex, Brave, Opera) по файлу настроек `Preferences`
- **Исправление спидометра сети:** Добавлен асинхронный пайплайн Mbps-прогресса в реальном времени, стрелка прибора больше не застывает на 0.0 Mbps во время скачивания
- **Исправление размера очистки:** Устранен баг подсчета байт в `nukeDirectoryContents` (теперь папки сканируются рекурсивно и возвращают точный освобожденный объем)
- **Исправление автозагрузки HKLM:** Опрос ключа реестра `StartupApproved\Run` теперь охватывает как `HKCU`, так и `HKLM` hives для корректного отображения статусов
- **Стабилизация Game Mode:** Процедура сжатия оперативной памяти `MinWorkingSet` переведена на щадящие рельсы (порог 200MB, защищен блоками try/catch)

## [1.7.0] - 2026-05-17

### Оптимизация производительности
- Внедрена персистентная PowerShell сессия для мониторинга оборудования (снижение CPU на 80%)
- Убран бесполезный `ProcessIdleTasks` из RAM Purge (был основан на мифе)
- Повышен порог MinWorkingSet с 50MB до 200MB (меньше hard page faults)
- Убраны PowerShell обёртки с `powercfg` в GameMode (экономия ~900ms)
- Заменён `splice` на `shift` в логгере (O(1) вместо O(n))
- Inline стили перенесены в CSS файлы для снижения CSSOM нагрузки

### Исправления багов
- Исправлен тест скорости интернета: заменён заблокированный `speed.cloudflare.com/__down` на Selectel/Hetzner/Tele2
- Исправлен toggle автозагрузки: HKLM элементы теперь записываются в правильный hive
- Исправлено измерение latency: HTTPS через Cloudflare вместо HTTP ip-api.com
- Результаты теста скорости показывают реальный сервер

### Офлайн поддержка
- Шрифты Inter (кириллица + латиница) встроены локально
- Material Symbols Outlined встроены локально (1100+ иконок)
- Удалены все CDN зависимости из index.html
- Приложение полностью работает без интернета

### Интерфейс
- Иконка трея: используется .ico для Windows (лучшая поддержка DPI)
- Обновлены иконки приложения
