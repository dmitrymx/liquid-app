# Changelog

Все заметные изменения в проекте документируются здесь.

## [1.7.0] - 2026-05-17

### Оптимизация производительности
- Внедрена персистентная PowerShell сессия для мониторинга оборудования (снижение CPU на 80%)
- Убран бесполезный `ProcessIdleTasks` из RAM Purge (был основан на мифе)
- Повышен порог MinWorkingSet с 50MB до 200MB (меньше hard page faults)
- Убраны PowerShell обёртки с `powercfg` в GameMode (экономия ~900ms)
- Заменён `splice` на `shift` в логгере (O(1) вместо O(n))
- Inline стили перенесены в CSS файлы для снижения CSSOM нагрузки

### Исправления багов
- Исправлен тест скорости интернета: заменён заблокированный `speed.cloudflare.com/__down` на Selectel/Hetzner/Tele2
- Исправлен toggle автозагрузки: HKLM элементы теперь записываются в правильный hive
- Исправлено измерение latency: HTTPS через Cloudflare вместо HTTP ip-api.com
- Результаты теста скорости показывают реальный сервер

### Офлайн поддержка
- Шрифты Inter (кириллица + латиница) встроены локально
- Material Symbols Outlined встроены локально (1100+ иконок)
- Удалены все CDN зависимости из index.html
- Приложение полностью работает без интернета

### Интерфейс
- Иконка трея: используется .ico для Windows (лучшая поддержка DPI)
- Обновлены иконки приложения
